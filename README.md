# CV Application

## 📌 Description
A React-based application that allows users to input and manage personal and professional information for a digital CV.

## 🚀 Features
- Form input handling
- Real-time data updates
- Component-based structure
- Editable sections

## 🛠️ Tech Stack
- React
- JavaScript
- CSS

## 📚 What I Learned
- React components and props
- Managing form state
- Handling user input
- Structuring reusable components
